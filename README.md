### Gendiff. 2th level Hexlet.
This utility compares two —Åonfiguration files (json or yaml) and generates a comparison document.
Three output formats are available.
### Project instalation: 
1. poetry install
2. poetry build
3. poetry publish --dry-run
4. python3 -m pip install  dist/*.whl
